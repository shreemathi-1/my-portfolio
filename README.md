# Shree Mathi · Portfolio

Lightweight, responsive single-page portfolio built with HTML , CSS, Bootstrap 5 and Javascript which displays profile, contacts, tech stack, projects, internships and achievements. Designed for easy customization and static hosting.

## Features
- Responsive layout (desktop / mobile)
- Clean, dark-card UI with soft accents
- Projects, skills and contact links
- Minimal JS (console greeting) — fully static HTML/CSS

## Tech stack
- HTML5
- CSS3 (custom styles)
- Bootstrap 5 (layout & utilities)
- Bootstrap Icons
- Google Fonts (Inter)
- Static hosting friendly (GitHub Pages, Netlify, Vercel)

## Local development
1. Clone the repo:
   git clone <repo-url>
2. Serve locally from project root (macOS):
   python3 -m http.server 8000
3. Open http://localhost:8000 in your browser.

## Deployment
- GitHub Pages: push to `main` (or use gh-pages branch) and enable Pages in repo settings.
- Netlify / Vercel: connect repo and deploy—no build step required.

## Customization
- Edit content in `index.html` (name, bio, projects, contacts).
- Replace `/images/photo.jpg` with your photo (same path or update src).
- Update colors and spacing in the <style> block or extract to a CSS file.
- Replace tech tags and badges to match your skills.

## Notes
- Keep external CDN links (Bootstrap, icons, fonts) for quick setup; self-host if offline or for privacy.
- Accessibility: ensure alt text and semantic updates as you add content.
